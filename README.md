# College Schedule API

API для информационно-справочной системы мониторинга учебного процесса

## Технологии
- ASP.NET Core 8.0
- Entity Framework Core
- PostgreSQL

## Структура проекта
- **Models** - сущности базы данных
- **Data** - контекст и конфигурация БД
- **Controllers** - API endpoints
- **Services** - бизнес-логика
- **DTO** - объекты передачи данных
